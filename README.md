  ## Description
  This app is for the use of creating and displaying the employees that are entered into the system via the command line. A HTML file will be generated after all questions have been answered in the  command line .

  ## Usage
  Start up the command line using node and answer all of the questions and the program will do the rest. Here is the link to the demonstration if you'd like to see how it runs.
